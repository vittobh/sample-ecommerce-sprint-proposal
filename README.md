# Sample — Ecommerce Sprint Proposal Template

Personal research sample from early PO / Scrum work. A complete sprint proposal template populated for a sample ecommerce web portal sprint.

See [SPRINT_PROPOSAL.md](SPRINT_PROPOSAL.md).

## What this sample shows
- Full sprint planning artifact (goal, dates, velocity, capacity)
- Pre-sprint / sprint / post-sprint hour accounting (dev + management)
- User-story estimation (story points + hours)
- Technical + management resource capacity model with leave deductions
- Roles & approval chain (PO, PM, PMO, CTO sponsor)

## Today's lens
A modern take would auto-generate the proposal from Jira / Linear backlog state, with AI-suggested story-point calibration based on historical velocity. See [vittobh/pm-os](https://github.com/vittobh/pm-os) for the agentic PM stack.
